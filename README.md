# Nurudeen Abdulsalaam — Portfolio

A modern React portfolio showcasing data science, machine learning, and analysis projects.

## Quick Start

```bash
npm install
npm run dev
```

## Managing Projects

- **Add**: Drop an image in `public/images/portfolio/`, add an entry to `src/data/projects.json`
- **Remove**: Delete the entry from `projects.json`
- **Push to GitHub** → auto-deploys via GitHub Actions

## Tech Stack

- **Vite + React** — fast builds, hot reload
- **Typed.js** — hero typing animation
- **Isotope** — animated project filtering
- **GitHub Pages** — free hosting, auto-deploy on push
