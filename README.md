## Managing Projects

All projects are data-driven — stored in a single JSON file for easy updates:

- **Add a project**: Drop an image in `public/images/portfolio/`, add an entry to `src/data/projects.json`
- **Remove a project**: Delete the entry from the JSON
- **Edit profile info**: Update `src/data/profile.json`
- **Push to GitHub** → auto-deploys via GitHub Actions
